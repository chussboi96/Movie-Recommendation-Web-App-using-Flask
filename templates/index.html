The project aims to create a web-based movie recommendation system that utilizes machine learning algorithms to suggest movies to users based on their search queries and preferences.

The application is built using the Flask framework in Python and is deployed locally on the userâ€™s system. It utilizes libraries such as Pandas and scikit-learn for data processing and machine learning algorithms.

  • The system takes user input and generates relevant movie suggestions by using a similarity matrix to find similar movies.
  • An interactive GUI is built to display recommended movies to users.
  • User-Profile management.

I used "The Movies Dataset," which is a collection of 5000 movies with credits data from the popular movie database TMDb.

Dataset

It contains information about the movies' title, genre, budget, release date, and popularity, among other features. I focused on 10 key features of the dataset.

Content-Based FIltering

The project uses a content-based filtering approach to recommend movies to users based on their search query and similarities between movie attributes.

4 types of recommendations:

1. Recommended movies based on the user's chosen genres and casts.

2. Most popular movies are based on different genres.

3. Most popular movies are based on different years.

4. Recommended movies similar to the user's selected movie.

Recommender System

The recommender system makes 10 recommendations to the user based on the type of content they watched.
